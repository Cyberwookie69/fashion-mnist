# Fashion MNIST - Liverpool ML assignment

Two-hidden-layer fully-connected neural network on Fashion MNIST,
plus comparisons across activation functions, dropout, optimizers,
classical baselines, and a learning-rate / epoch / batch-size sweep.

## Run on Colab

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/Cyberwookie69/fashion-mnist/blob/main/Fashion_MNIST_Colab.ipynb)

## What is here

| File | Purpose |
|---|---|
| `fashion_mnist.py` | Main script: Parts 1-3 + 4 extensions. Produces all plots and `results_summary.md`. |
| `matrix_sweep.py` | Grid sweep over (epochs x batch_size x activation x dropout). 1300 cells, ~2h locally. |
| `plot_sigmoid_3d.py` | 3D wiremesh of sigmoid test_acc across (E, B). |
| `plot_relu_3d.py` | Same for ReLU. |
| `plot_extended_sigmoid.py` | Sigmoid sweep extended to E=1280 (~60-90 min). |
| `view_ubyte.py` | Sanity-check the IDX/ubyte data files. |
| `Fashion_MNIST_Colab.ipynb` | Notebook entry point for Colab. |

## Run locally

Requirements: Python 3.10+, TensorFlow 2.x, numpy, pandas, matplotlib, scikit-learn, Pillow, tqdm.

```powershell
python fashion_mnist.py
```

The data lives in `archive (1)/` as the original IDX/ubyte files
(`train-images-idx3-ubyte` etc.), which Keras' `fashion_mnist.load_data()`
also produces if you don't have them locally.
